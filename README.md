# monitorDashboard-back 
Monitor dashboard backend.

## Tecnologias
- Java 8
- Spring Boot 2.3.1
- FlyWay
- MySQL 5

## Como executar
- Na pasta raiz, executar:

```bash
docker-compose up --build
```

## Como testar
- Na pasta raiz, executar:

```bash
./mvnw verify
```
- Abrir o arquivo `target/site/jacoco/index.html`.
